GIT

Git is the free and open source distributed version control system that's responsible for everything git hub related happens locally on the computer

GIT COMMANDS

  1. git init - initialise an existing directory
  2. git log - show the commit history for active branch
  3. git stash - save modified and staged changes

VI

The default editor that comes with the UNIX operating system is called the vi(visual editor)

VI COMMANDS

  1. vi filename - To edit the file name
  2. :x - To quit the vi
  3. :wq - quit vi and write the modified filke to file names in original location
  4. i - insert text before cursor

LINUX

The Linux is a community of open-source Unix like operating system that are based on thge Linus Kernel. It is a free and open-source operating system and source code can be modified

LINUX COMMANDS

  1. ls - directory listing
  2. cd - change directory
  3. pwd - show the present directory
  4. mkdir - create directory
